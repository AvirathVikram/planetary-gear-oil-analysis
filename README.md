# CFD Analysis of Lubrication Flow in a Planetary Gear System
(Ongoing - Research Project under DRDO-funded study)

## Overview
This project focuses on analyzing lubrication flow behavior inside a planetary gear system using CFD tools. The objective is to understand oil distribution and identify regions of poor lubrication.

## Tools Used
- ANSYS (Fluent/CFX)
- CAD (SolidWorks / CATIA)

## Work Done So Far
- Developed simplified planetary gear geometry
- Defined inlet and outlet boundary conditions for lubricant flow
- Set up initial simulation framework in ANSYS

## Current Focus
- Studying flow distribution across gear regions
- Identifying potential stagnation zones

## Future Work
- Incorporating gear motion into the simulation
- Performing transient analysis
- Mesh refinement and validation
